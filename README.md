# Jリーグの新卒選手獲得の動向調査(**J** League **Ne**wcomers **T**rends **S**urvey)
Jリーグの新卒選手獲得の動向調査と題し、[Soccer D.B.](https://soccer-db.net)様のデータを基に新卒選手のキャリア形成を分析するデータセットを作成いたしました。以下に概要を記載します。

- 2017~2023年までの新卒選手の各種データを集めました。
    - 高校、大学から直接Jリーグにプロ入りした選手が対象です。
    - 以下のデータについて収集しました。
        - 高卒か？大卒か？（type:"高卒","大卒",type_id:1,2）
        - 高校のチームの種別（category:"ユース","高体連","街クラブ",category_id:"1","2","3"）
        - ポジション
        - 高校年代及び大学年代での所属チーム、入団先チームの情報
            - チーム名
            - 地域名
            - 都道府県名
        - 選手のプレイログ
            - 2023年までの各種大会(J1,J2,インターハイ等)での出場実績
- 参照元に無い情報（特に高校の情報）は独自で追加いたしました。
- データ形式はjsonで上と同じ階層になっています。
